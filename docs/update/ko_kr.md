    * 사용법 예 (1): `foo = def(a,b,c, a+b+c)`
    * 사용법 예 (2)
    ```
    foo_return = x + y + z
    foo = def(x, y, z, foo_return)
    
    a = foo(1, 2, 3)
    ```
    * 단, 반환변수가 매우 불안정합니다.
2. 이제, `print` 메소드가 값의 코드도 출력합니다.

    * `a = foo(..., a)` 꼴이 정상적인 값을 출력하지 않습니다.
